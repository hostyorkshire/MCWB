# Logs Directory

This directory contains log files generated by the MCWB Weather Bot and MeshCore system.

## Log Files

- `weather_bot.log` - General application log (info, debug)
- `weather_bot_error.log` - Error log (errors and critical issues only)
- `meshcore.log` - MeshCore library log (debug output)

## Log Rotation

Logs are automatically rotated when they reach 10 MB, keeping up to 5 backup files.

## Viewing Logs

```bash
# View latest weather bot log
tail -f logs/weather_bot.log

# View latest errors
tail -f logs/weather_bot_error.log

# View last 100 lines
tail -n 100 logs/weather_bot.log

# Search for specific errors
grep -i "error" logs/weather_bot.log
```

## Log Cleanup

Old log files are automatically removed by the rotation system. To manually clean up:

```bash
# Remove old log files
rm logs/*.log.*

# Remove all logs
rm logs/*.log
```

## Log Levels

- **DEBUG**: Detailed information for debugging
- **INFO**: General informational messages
- **WARNING**: Warning messages (non-critical issues)
- **ERROR**: Error messages (failures that don't stop the bot)
- **CRITICAL**: Critical issues (severe failures)
